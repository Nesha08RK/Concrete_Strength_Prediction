## Problem Statement

Concrete compressive strength is a critical parameter in construction. Traditional testing methods require laboratory experiments and curing periods of up to 28 days, making the process time-consuming and costly. This project aims to develop a machine learning-based system to predict concrete compressive strength and support sustainable concrete mix design.

## Objectives

* Predict concrete compressive strength using machine learning.
* Compare the performance of different ML models.
* Provide explainable predictions using SHAP.
* Develop a user-friendly prediction system.

## Research Gap

* Existing studies focus mainly on strength prediction.
* Cost and carbon emission analysis are rarely considered.
* Lack of concrete mix recommendation systems.
* Limited support for sustainable construction decisions.

## Novelty

* Concrete Mix Recommendation System.
* Cost Estimation Module.
* Carbon Emission Analysis.
* SHAP-Based Explainable AI.
* Multi-Objective Decision Support.
